# 读书笔记生成命令使用说明

## 快速使用

在 Claude Code 中使用以下命令生成读书笔记：

```
读书笔记 <书名>
```

### 示例

```
读书笔记 重构
读书笔记 "设计模式"
读书笔记 "代码整洁之道"
```

## 命令参数

| 参数 | 说明 | 必需 | 示例 |
|------|------|------|------|
| 书名 | 要生成笔记的书籍名称 | 是 | `重构`、`"设计模式"` |
| 详细程度 | 笔记详细程度（可选） | 否 | `simple`、`medium`、`detailed` |

## 详细程度说明

- **simple** - 简要版：基础信息、内容概要、简短笔记
- **medium** - 中等版：包含知识架构图、分章节笔记、基础代码示例（默认）
- **detailed** - 详细版：完整知识架构、深入笔记、大量代码示例、实践案例

## 输出位置

笔记文件将生成在：
```
/mnt/c/Users/qq691/Desktop/readbooks/books/<书名>/<书名>.md
```

## 笔记模板结构

```
# 📚 <书名>读书笔记

## 📖 基本信息
## 📝 内容概要
## 🧠 知识架构（Mermaid 思维导图）
## ✍️ 读书笔记（分章节详细笔记）
## 💡 个人思考
## 🎯 实践应用
## 🔗 相关扩展
## 📊 学习总结
## 📈 阅读进度
```

## 完整示例

```
读书笔记 "重构：改善既有代码的设计" detailed
```

这将：
1. 搜索书籍信息
2. 创建 `books/重构：改善既有代码的设计/` 目录
3. 生成详细的读书笔记，包含：
   - 完整的书籍元数据
   - 详细的 Mermaid 知识架构图
   - 分章节的深入笔记
   - JavaScript 代码示例
   - 个人思考和实践计划
   - 相关资源链接

## 注意事项

1. 书名如果包含空格，请用引号括起来
2. 如果书籍已有笔记，会询问是否覆盖
3. 生成后会显示文件路径和预览
4. 支持中文和英文书名

## 高级用法

### 批量生成

```
# 一次生成多本书的笔记
读书笔记 "重构" && 读书笔记 "设计模式" && 读书笔记 "代码整洁之道"
```

### 指定作者（提高搜索准确性）

```
读书笔记 "Clean Architecture" author:"Robert C. Martin"
```
