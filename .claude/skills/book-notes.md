# 读书笔记生成

生成标准化的读书笔记，适用于 `readbooks` 项目。

## 参数

- `bookName`: 书名（必需）
- `author`: 作者（可选，会自动搜索）
- `detailLevel`: 详细程度（可选，默认为 `medium`，可选值：`simple`、`medium`、`detailed`）

## 示例

```
读书笔记 重构
读书笔记 "设计模式"
读书笔记 "代码整洁之道" detailed
```

## 模板路径

- 模板文件：`/mnt/c/Users/qq691/Desktop/readbooks/templates/读书笔记模板.md`
- 笔记目录：`/mnt/c/Users/qq691/Desktop/readbooks/books/`

## 输出格式

生成的读书笔记包含以下部分：

1. **📚 基础信息** - 书名、作者、出版社、出版年份、标签等
2. **📖 内容概要** - 书籍简介、核心主题、主要章节
3. **🧠 知识架构** - Mermaid 思维导图
4. **✍️ 读书笔记** - 分章节的详细笔记，包含代码示例
5. **💡 个人思考** - 阅读过程中的思考和感悟
6. **🎯 实践应用** - 行动计划和实践建议
7. **🔗 相关扩展** - 相关书籍和在线资源
8. **📊 学习总结** - 收获、观念改变、未来行动
9. **📈 阅读进度** - 章节进度追踪

## 执行步骤

1. 搜索书籍信息（WebSearch）
2. 创建书籍目录（如不存在）
3. 生成符合模板的读书笔记
4. 包含 Mermaid 知识架构图
5. 根据详细程度生成相应内容
