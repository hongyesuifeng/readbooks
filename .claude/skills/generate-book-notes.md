# 生成读书笔记

为指定的书籍生成详细完整的读书笔记。

## 使用方法

```
/skill 生成读书笔记 <书名>
```

## 示例

```
/skill 生成读书笔记 重构
/skill 生成读书笔记 "设计模式"
/skill 生成读书笔记 "Clean Code"
```

## 执行步骤

1. 使用 WebSearch 搜索书籍信息
2. 读取笔记模板了解格式
3. 参考现有笔记风格
4. 生成详细笔记（包含 Mermaid 图、代码示例、个人思考等）
5. 保存到 books/<书名>/<书名>.md
6. 显示文件路径和预览

## 输出位置

```
books/<书名>/<书名>.md
```
