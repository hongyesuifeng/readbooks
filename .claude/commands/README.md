# 读书笔记生成命令使用说明

## 快速使用

在 Claude Code 中使用以下命令生成读书笔记：

```
/book-notes <书名>
```

### 示例

```
/book-notes 重构
/book-notes "设计模式"
/book-notes "代码整洁之道"
/book-notes "Clean Architecture"
```

## 命令说明

| 命令 | 说明 | 语言 |
|------|------|------|
| `/book-notes` | 生成读书笔记 | 中文输出 |

## 工作流程

```
输入命令
    ↓
搜索书籍信息
    ↓
读取笔记模板
    ↓
生成详细笔记
    ↓
保存文件
    ↓
显示预览
```

## 注意事项

1. **书名格式**：书名包含空格时请用引号括起来
2. **文件覆盖**：如果笔记已存在，会询问是否覆盖
3. **网络搜索**：需要网络连接来搜索书籍信息
4. **代码语言**：技术书籍默认使用 JavaScript 代码示例
5. **内容语言**：笔记内容为中文，技术术语保留英文

## 常见问题

**Q: 如何查看已生成的笔记？**
```
笔记保存在 books/<书名>/<书名>.md
你可以用任何 Markdown 编辑器打开查看
```

**Q: 如何更新已有笔记？**
```
重新运行命令即可，会询问是否覆盖
```

**Q: 可以自定义笔记模板吗？**
```
可以，编辑 templates/读书笔记模板.md
```

**Q: 生成的笔记可以手动编辑吗？**
```
完全可以！笔记生成后你可以随意编辑、添加个人笔记
```
