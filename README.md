# ReadBooks 📚

> 个人读书笔记与学习记录项目

## 项目简介

ReadBooks 是一个用于记录读书笔记、整理知识体系的个人学习项目。每读一本书，我都会：
- 📖 生成详细的读书笔记
- 🧠 绘制知识架构图
- 🔍 深度解析书籍内容
- 💬 记录个人思考和感悟

## 项目结构

```
readbooks/
├── books/           # 读书笔记存放目录
├── templates/       # 笔记模板
├── assets/          # 图片资源
├── 项目架构.md      # 项目架构说明
└── CLAUDE.md        # AI助手指导文件
```

## 读书笔记特色

每本书的读书笔记包含：
- 📊 **内容概要** - 书籍简介、核心主题、章节概述
- 🧠 **知识架构** - Mermaid思维导图展示知识体系
- ✍️ **读书笔记** - 重点摘录、个人思考、实践应用
- 🔗 **相关扩展** - 相关书籍推荐、延伸阅读

## 使用方法

1. **开始阅读新书** - 告诉我你要读的书名
2. **自动生成笔记** - 我会搜索网络信息生成初始笔记
3. **阅读过程记录** - 阅读中随时补充个人感悟
4. **完成知识架构** - 生成整本书的知识体系图
